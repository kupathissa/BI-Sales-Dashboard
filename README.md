# HR Analytics Dashboard -- Power BI

## Project Overview

This project presents an interactive HR analytics dashboard built using
Microsoft Power BI. The dashboard provides insights into employee hiring
trends, workforce demographics, and key HR performance indicators.

## Features

-   New Hire Trends (Current Year vs Previous Year)
-   Active Employee Analysis
-   Employee Distribution by Region
-   Workforce Breakdown by Gender
-   Workforce Breakdown by Age Group
-   Bad Hire Analysis (\<60 days employment)
-   Hiring Trend Analysis by Month
-   Interactive filters and drill‑down capabilities

## Dashboard Visuals

1.  KPI Cards -- Total New Hires, Active Employees
2.  Combo Chart -- Monthly New Hire Count vs Last Year
3.  Stacked Bar Charts -- Employees by Region and Ethnicity
4.  Pie/Donut Charts -- Gender and Age Distribution
5.  Waterfall Chart -- Bad Hires Percentage by Age Group
6.  Bar Chart -- Active Employees by Region

## Tools & Technologies

-   Microsoft Power BI Desktop
-   DAX (Data Analysis Expressions)
-   Power Query
-   Data Modeling

## Insights

-   Hiring peaks in mid‑year months.
-   Some regions show higher employee concentration.
-   Younger age groups (\<30) represent a large share of new hires.
-   Early turnover can be monitored using the Bad Hire metric.

## How to Use

1.  Download the `.pbix` file.
2.  Open it in Power BI Desktop.
3.  Connect or replace the dataset.
4.  Explore the interactive dashboard visuals.
